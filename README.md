# Wordle MEM Solver

A small research-oriented project on a modified entropy-based Wordle solver.

This repository contains a technical report, LaTeX sources, an experiment notebook, and a Windows executable demo for **Modified Entropy Maximizing (MEM)**, a Wordle solving strategy that analyzes the trade-off between **partition diversity** and **partition uniformity**.

## Overview

Wordle can be viewed as an information search problem.
For each guess, the remaining candidate answers are divided into feedback-pattern groups. A good guess is one that reduces the candidate set effectively after receiving feedback.

Standard entropy-based Wordle solvers evaluate a guess by the entropy of its feedback distribution. However, entropy combines two different properties into a single value:

1. **Partition diversity** — how many non-empty feedback blocks are created.
2. **Partition uniformity** — how evenly candidates are distributed among those blocks.

This project separates these two factors and combines them using a tunable score called **Modified Entropy Maximizing (MEM)**.

## MEM Score

For a guess `g`, let:

- `C` be the current candidate answer set.
- `n = |C|`.
- `k` be the number of non-empty feedback blocks generated by `g`.
- `H(g)` be the entropy of the feedback distribution.

The partition diversity score is defined as:

```text
B(g) = log2(k) / log2(min(243, n))
```

The partition uniformity score is defined as:

```text
U(g) = H(g) / log2(k)
```

The MEM score is:

```text
Mλ(g) = B(g)^(2λ) · U(g)^(2(1 - λ))
```

where `λ ∈ [0, 1]` controls the balance between diversity and uniformity.

- `λ = 0.5` corresponds to normalized standard entropy.
- `λ > 0.5` gives more weight to partition diversity.
- `λ < 0.5` gives more weight to partition uniformity.

## Key Results

The experiment was performed using:

- 2,309 Wordle answer candidates
- 12,947 allowed guess words
- λ values from `0.0` to `1.0` in increments of `0.1`

| λ | First Guess | Mean Attempts | Max Attempts |
|---|---|---:|---:|
| 0.0 | roate | 5.0645 | 9 |
| 0.5 | soare | 3.4638 | 6 |
| 0.7 | reast | 3.4353 | 5 |
| 1.0 | trace | 3.4305 | 6 |

The best average-case performance was obtained with `λ = 1.0`, which achieved a mean of **3.4305 attempts**.

However, `λ = 0.7` showed a more balanced distribution: it solved every answer within **5 attempts** and achieved the highest number of answers solved within 4 attempts.

These results suggest that emphasizing partition diversity can improve the average performance of an entropy-based Wordle solver, while a moderate balance between diversity and uniformity can improve stability.


If the notebook is renamed in the future, update the command accordingly.
